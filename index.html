SaaS CRM管理画面/toCアプリ画面開発

主に飲食や小売店等の事業者向けに、スタンプカードなどのCRM販促ツールを提供するSaaSを提供。toC向けに、決済サービスの利用者がクーポンや特典を利用できるシステムの構築を行った。

環境・言語

React/Redux, Java Spring, AWS

自身の担当

画面デザインをもとに画面項目定義を作成、デザイナー／企画と出来上がったデザインに対してのUIの調整、apiI/F設計、フロントエンドアーキテクチャ設計、開発環境構築を行っている。事業者向け販促ツール画面はReact/Redux、決済サービス利用者が利用するアプリはアプリ内のwebview画面をReactで開発予定。フロントエンド側のリーダーというポジションでもあるので、PLと連携して必要な工数の割り出し、チケット(JIRA)作成やメンバーへのフォローも行っている。
